Object-Oriented Educational Engine ("Mr. H")

Type: Academic Project (OOP Architecture)
Role: Core Mechanics & State Logic
Tech Stack: TypeScript, ES6 Modules, HTML5 Canvas
Repository: [ View Source ]

"Mr. H" is an interactive, web-based educational game built from scratch to teach cybersecurity fundamentals and password safety to younger demographics. The project was an exercise in applying strict Object-Oriented Programming (OOP) paradigms and ES6 modules without relying on pre-built game engines.

Technical Implementation

  • Core Mechanics & Physics: Developed the underlying collision detection engine from the ground up and implemented localized, object-driven minigame logic.
  • State Management: Co-engineered the application's state transitions and scene routing, finalizing the logic loop between the Start, Regional, and End phases to ensure seamless memory management between states.
  • Agile Crisis Recovery: Spearheaded project recovery during a critical mid-cycle team restructuring. Rapidly reassessed the codebase, redistributed technical tasks, and increased code velocity to ensure a successful final deployment.
